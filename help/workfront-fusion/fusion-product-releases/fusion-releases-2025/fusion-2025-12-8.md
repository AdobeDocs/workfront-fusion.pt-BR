# Atividade de lançamento do Workfront Fusion: semana de 8 de dezembro de 2025

Esta página descreve todos os aprimoramentos realizados no Adobe Workfront Fusion na semana de 8 de dezembro de 2025.

Para obter uma lista de todas as alterações recentes, consulte a [atividade de lançamento do Adobe Workfront Fusion](/help/workfront-fusion/fusion-product-releases/fusion-release-activity.md).

Para obter uma lista de correções de erros recentes no Workfront Fusion, consulte a página [Atualizações de manutenção do Workfront](https://experienceleague.adobe.com/pt-br/docs/workfront-known-issues/releases/current-updates) e verifique se há atualizações rotuladas como atualização de manutenção do Workfront Fusion.

## O conector do Adobe InDesign já está disponível

Para facilitar o uso do Adobe InDesign com o Fusion, criamos um conector exclusivo do Adobe InDesign. Com os módulos do Adobe InDesign, você pode:

* Criar representações
* Listar ou obter detalhes de scripts personalizados
* Remapear links
* Mesclar dados ou obter tags de mesclagem de dados
* Obter informações de documentos
* Enviar uma solicitação de execução de script personalizado
* Fazer uma chamada personalizada à API do InDesign

Para mais informações, consulte os [módulos do Adobe InDesign](/help/workfront-fusion/references/apps-and-modules/adobe-connectors/indesign-modules.md).
