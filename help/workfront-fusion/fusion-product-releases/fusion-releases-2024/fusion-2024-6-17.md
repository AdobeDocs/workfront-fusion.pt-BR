# Atividade de lançamento do Workfront Fusion: semana de 17 de junho de 2024

Esta página descreve todos os aprimoramentos realizados no Adobe Workfront Fusion na semana de terça-feira, 17 de junho de 2024.

Para obter uma lista de todas as alterações recentes, consulte a [atividade de lançamento do Adobe Workfront Fusion](/help/workfront-fusion/fusion-product-releases/fusion-release-activity.md).

Para obter uma lista de correções de erros recentes no Workfront Fusion, consulte a página [Atualizações de manutenção do Workfront](https://experienceleague.adobe.com/pt-br/docs/workfront-known-issues/releases/current-updates) e verifique se há atualizações rotuladas como atualização de manutenção do Workfront Fusion.

## Conector e módulos do Adobe I/O Events agora disponíveis

Para permitir a integração com o Adobe I/O Events, criamos um novo conector. Com os módulos do Adobe Adobe I/O Events, você pode:

* Criar um registro de evento
* Listar todos os eventos em um diário
* Obter provedor ou IDs de evento
* Fazer uma chamada de API personalizada para a API do Adobe Journey Optimizer

Para obter informações sobre o conector do Adobe Journey Optimizer, consulte [[!DNL Adobe I/O Events] módulos](/help/workfront-fusion/references/apps-and-modules/adobe-connectors/adobe-io-events-modules.md).

## Gerar várias imagens no conector do Adobe Firefly

Gerar várias imagens permite selecionar aquela que melhor corresponde ao que você está procurando. Agora, os módulos do Adobe Firefly podem gerar até quatro imagens por execução de módulo.

Anteriormente, os módulos do só podiam gerar uma única imagem por execução.

Para obter informações sobre o conector do Adobe Firefly, consulte [módulos do Adobe Firefly](/help/workfront-fusion/references/apps-and-modules/adobe-connectors/adobe-firefly-modules.md).
