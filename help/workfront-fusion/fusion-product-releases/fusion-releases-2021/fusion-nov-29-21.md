# Atividade de lançamento do Workfront Fusion: semana de terça-feira, 29 de novembro de 2021

Esta página descreve todos os aprimoramentos realizados no Adobe Workfront Fusion na semana de terça-feira, 29 de novembro de 2021.

Para obter uma lista de todas as alterações recentes, consulte a [atividade de lançamento do Adobe Workfront Fusion](/help/workfront-fusion/fusion-product-releases/fusion-release-activity.md).

Para obter uma lista de correções de erros recentes no Workfront Fusion, consulte a página [Atualizações de manutenção do Workfront](https://experienceleague.adobe.com/pt-br/docs/workfront-known-issues/releases/current-updates) e verifique se há atualizações rotuladas como atualização de manutenção do Workfront Fusion.

## Serviços da Adobe PDF agora disponíveis no Workfront Fusion

Agora é possível usar o Adobe Workfront Fusion para criar e modificar PDFs. Como o Workfront Fusion faz parte do conjunto de produtos da Adobe, você não precisa de uma conta separada do Adobe para usar esses módulos.

Com os módulos de ferramentas do Adobe PDF, você pode:

* Gerar uma PDF a partir dos dados fornecidos
* Extrair dados de uma PDF
* Criar uma PDF a partir de uma imagem, documento ou arquivo HTML
* Conversão de uma PDF em uma imagem ou documento
* Combinar arquivos PDF em um arquivo PDF maior
* Proteger um arquivo PDF com uma senha ou remover a proteção por senha de uma PDF
* Compactar arquivos PDF para usar menos memória ou largura de banda
* Linearizar um arquivo do PDF para que ele possa ser lido página por página antes que o arquivo inteiro seja baixado
* Usar o OCR (Optical Character Recognition, reconhecimento óptico de caracteres) em um arquivo
* Modificar uma PDF excluindo ou girando determinadas páginas
* Obter propriedades de arquivos do PDF, como contagem de páginas, versão ou criptografia
