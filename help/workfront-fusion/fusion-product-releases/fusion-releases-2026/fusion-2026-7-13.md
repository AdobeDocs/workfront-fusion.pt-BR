# Atividade de lançamento do Workfront Fusion: semana de 13 de julho de 2026

Esta página descreve todas as melhorias feitas no Adobe Workfront Fusion na semana de 13 de julho de 2026.

Para obter uma lista de todas as alterações recentes, consulte a [atividade de lançamento do Adobe Workfront Fusion](/help/workfront-fusion/fusion-product-releases/fusion-release-activity.md).

Para obter uma lista de correções de erros recentes no Workfront Fusion, consulte a página [Atualizações de manutenção do Workfront](https://experienceleague.adobe.com/pt-br/docs/workfront-known-issues/releases/current-updates) e verifique se há atualizações rotuladas como atualização de manutenção do Workfront Fusion.

## Conector do Workfront Planning versão 2 disponível.

Para aproveitar os recursos expandidos na API do Workfront Planning versão 2, criamos um novo conector. Com os módulos do Workfront Planning v2, é possível:

* Criar, ler, atualizar e excluir espaços de trabalho, tipos de registro. registros e campos.
* Obter informações como ID de autorização e permissões para um usuário.
* Solicite acesso a um recurso ou visualize as solicitações de acesso pendentes.
* Fazer uma chamada de API personalizada para a API do Workfront Planning v2.

Para obter informações e instruções, consulte [Módulos do Workfront Planning](/help/workfront-fusion/references/apps-and-modules/adobe-connectors/workfront-planning-modules.md).

## Usar mTLS em webhooks personalizados

Para garantir que os webhooks tenham mais necessidades de segurança, adicionamos a capacidade de usar mTLS em webhooks personalizados. Agora, você pode inserir um certificado de cliente e uma chave pública no webhook, que usará essas credenciais para se conectar ao serviço.

Para obter mais informações sobre como configurar webhooks, consulte [Webhooks](/help/workfront-fusion/references/apps-and-modules/universal-connectors/webhooks-updated.md).
