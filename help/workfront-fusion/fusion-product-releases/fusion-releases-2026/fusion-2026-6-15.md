# Atividade de lançamento do Workfront Fusion: semana de 15 de junho de 2026

Esta página descreve todas as melhorias feitas no Adobe Workfront Fusion na semana de 15 de junho de 2026.

Para obter uma lista de todas as alterações recentes, consulte a [atividade de lançamento do Adobe Workfront Fusion](/help/workfront-fusion/fusion-product-releases/fusion-release-activity.md).

Para obter uma lista de correções de erros recentes no Workfront Fusion, consulte a página [Atualizações de manutenção do Workfront](https://experienceleague.adobe.com/pt-br/docs/workfront-known-issues/releases/current-updates) e verifique se há atualizações rotuladas como atualização de manutenção do Workfront Fusion.

## Futura descontinuação de módulos herdados do Photoshop

O Adobe Photoshop descontinuou alguns elementos de sua API, que o Fusion usa para executar ações no Photoshop.

**Portanto, alguns dos módulos existentes do Photoshop não funcionarão após 30 de julho de 2026.**

Recomendamos atualizar todos os cenários que usam esses módulos para os módulos atualizados o mais rápido possível.

Para obter uma lista de módulos afetados, consulte [atualizações de descontinuação da API do Adobe Photoshop](/help/workfront-fusion/references/apps-and-modules/adobe-connectors/adobe-photoshop-modules.md#adobe-photoshop-api-deprecation-updates).

Para obter uma explicação de como as alterações na API afetam o Workfront Fusion, consulte [Visão geral das APIs no Fusion](/help/workfront-fusion/get-started-with-fusion/understand-fusion/api-overview.md).


## Novos endpoints agora disponíveis para a API do Adobe Workfront Fusion

Para tornar a API do Adobe Workfront Fusion mais flexível e eficiente, adicionamos o endpoint para as seguintes ações:

* **Cenários**: listar cenários, obter cenários por ID, clonar um cenário, exportar blueprints e obter dependências de cenário
* **Execuções**: listar a execução e obter uma única execução
* **Conexões**: liste conexões, obtenha uma conexão por ID e liste cenários usando uma conexão
* **Repositórios de dados**: liste os repositórios de dados por equipe e obtenha um repositório de dados por ID
* **Pastas**: listar, criar, atualizar e excluir pastas
* **Webhooks**: liste webhooks e obtenha um webhook por ID

Para exibir o changelog completo, consulte [Changelog](https://developer.adobe.com/workfront-fusion-apis/getting-started/changelog/) na documentação do desenvolvedor do Workfront Fusion.

## Conector de análise unificada e aprovações do Adobe Workfront agora disponível

Para expandir a capacidade do Fusion de trabalhar com revisão e aprovações unificadas no Workfront, criamos o conector de Revisão e Aprovações unificadas do Adobe Workfront. Com a análise unificada e aprovações da Adobe Workfront:

* Criar, atualizar ou excluir aprovações e estágios
* Adicionar ou lembrar participantes da aprovação
* Tomar ou excluir decisões
* Recuperar informações sobre aprovações, modelos ou bots de aprovação
* Obter aprovações sugeridas para os participantes
* Fazer uma solicitação de API personalizada

Para obter informações e instruções, consulte os [módulos de Revisão e Aprovações Unificadas do Adobe Workfront](/help/workfront-fusion/references/apps-and-modules/adobe-connectors/workfront-review-and-approvals-modules.md).

## Novos módulos para o conector do Adobe Photoshop

Para expandir os recursos do conector do Adobe Photoshop, adicionamos estes módulos:

* Criar uma prancheta
* Criar ou editar um composto
* Edição de uma imagem com vários ajustes
* Executar ações, scripts e transformações do Photoshop
* Gerar um manifesto

Para obter informações e instruções para os novos módulos, consulte [módulos do Adobe Photoshop](/help/workfront-fusion/references/apps-and-modules/adobe-connectors/adobe-photoshop-modules.md).

## Novos módulos disponíveis para o conector do Adobe InDesign

Para expandir os recursos do conector do Adobe InDesign, adicionamos estes módulos:

* Enviar um script personalizado
* Atualizar versão do aplicativo de script personalizado
* Obter versões atuais do aplicativo
* Converter PDF em documento do InDesign

Para obter informações e instruções para os novos módulos, consulte [módulos do InDesign](/help/workfront-fusion/references/apps-and-modules/adobe-connectors/indesign-modules.md).
