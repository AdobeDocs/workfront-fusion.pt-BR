# Atividade de lançamento do Workfront Fusion: semana de terça-feira, 9 de fevereiro de 2026

Esta página descreve todos os aprimoramentos realizados no Adobe Workfront Fusion na semana de terça-feira, 9 de fevereiro de 2026.

Para obter uma lista de todas as alterações recentes, consulte a [atividade de lançamento do Adobe Workfront Fusion](/help/workfront-fusion/fusion-product-releases/fusion-release-activity.md).

Para obter uma lista de correções de erros recentes no Workfront Fusion, consulte a página [Atualizações de manutenção do Workfront](https://experienceleague.adobe.com/pt-br/docs/workfront-known-issues/releases/current-updates) e verifique se há atualizações rotuladas como atualização de manutenção do Workfront Fusion.

## Novo Microsoft SharePoint Online > Criar um módulo de arquivo

Adicionamos um novo módulo Criar um arquivo ao conector do Microsoft SharePoint Online. O novo módulo oferece melhor desempenho ao trabalhar com arquivos grandes.

O módulo Criar um arquivo disponível anteriormente foi renomeado como Criar um arquivo (Herdado). Embora o módulo herdado ainda esteja disponível, recomendamos usar o novo módulo.

Para obter mais informações sobre módulos do Microsoft SharePoint Online, consulte [módulos do Microsoft SharePoint Online](/help/workfront-fusion/references/apps-and-modules/third-party-connectors/sharepoint-modules.md).
