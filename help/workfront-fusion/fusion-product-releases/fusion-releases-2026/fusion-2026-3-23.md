## Disponibilizar um módulo de chamada de API personalizada (Polling) no Adobe Authenticator Connector

Para possibilitar a repetição de chamadas de API até que uma determinada condição ou limite seja atendido, criamos o módulo Fazer uma chamada de API personalizada (Polling). Este módulo inclui opções que permitem configurar chamadas de API repetidas. Você pode fazer com que a chamada de API seja executada até que uma determinada condição seja atendida, definir um número específico de tentativas e configurar o tempo entre as tentativas.

Para obter mais informações, consulte [Fazer uma chamada de API personalizada (Polling)](/help/workfront-fusion/references/apps-and-modules/adobe-connectors/adobe-authenticator-modules.md#make-a-custom-api-call-polling) no artigo Módulos do Adobe Authenticator.
